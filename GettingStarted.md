# Get set up: Cursor

We’ll use [Cursor](https://cursor.com) as our editor — an AI-native code editor that’s great for iterating quickly and trying ideas without leaving the flow.

## Download & install

1. **Download**  
   Go to **[cursor.com](https://cursor.com)** and click **Download**.  
   Choose the version for your OS (macOS, Windows, or Linux).

2. **Install**  
   - **macOS:** Open the downloaded `.dmg`, drag Cursor into Applications.  
   - **Windows:** Run the installer and follow the prompts.  
   - **Linux:** Use the provided `.AppImage` or install via the deb/rpm package if available.

3. **Open Cursor**  
   Launch Cursor like any other app. Sign in or create a free account when prompted (needed for AI features).

4. **Open this repo**  
   In Cursor: **File → Open Folder** and select your local `DanceHack` (or `Dance_Hack_Vibe_Coding`) folder. You’re ready to vibe code.
